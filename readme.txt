== Description ==

Add Screenr to the available oEmbed providers

Adds the ability to paste a video URL from Screenr.com and have it automatically embed into a WordPress blog post, page or custom post type.

Simply paste the Screenr.com video URL into the content area and I will oEmbed into your post, page or custom post type.

Example:

	http://www.screenr.com/a3KN

== Installation ==

1. Go to Dashboard >> Plugins >> Add New >> Search
2. Search for "Screenr oEmbed," click "Install Now," then "Activate Plugin"
3. There are no settings

== Frequently Asked Questions ==

= What is Screenr? =

Instant screencasts. Just login at <a href="http://screenr.com">Screenr.com</a> and create a screencast, up to 5 minutes with the free version

= Where are the settings? =

There are no settings. Activating Screenr oEmbeds enables oEmbed support for Screenr.com videos for all users.
